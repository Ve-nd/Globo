# Globo — All You Want

A production-quality interactive demo of a delivery platform built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion.

## 4 Apps in One

- 🍔 **Customer App** — Browse stores, add to cart, checkout, live order tracking, reviews, profile
- 🏪 **Merchant Dashboard** — KPI analytics, menu management, live order kanban, store settings
- 🛵 **Driver App** — Available orders, turn-by-turn navigation, delivery status, earnings tracker
- ⚙️ **Admin Dashboard** — City/radius management, merchant moderation, driver fleet, platform settings

## Tech Stack

- **React 18 + TypeScript** — type-safe components
- **Vite** — lightning-fast dev server & build
- **Tailwind CSS v3** — utility-first styling with custom Globo palette
- **Framer Motion** — smooth page transitions and micro-animations
- **React Router DOM v6** — client-side routing
- **Zustand** — global cart state management
- **React Hot Toast** — toast notifications

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
